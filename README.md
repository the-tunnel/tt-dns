# tt-dns
Yet another clean dns forwarder

----
### Usage
```
TT-DNS 0.1.1
Yet another clean dns forwarder

USAGE:
    tt-dns [OPTIONS]

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -l, --listen <listen>         [default: 127.0.0.1:53]
    -u, --upstream <upstream>     [default: 8.8.8.8:53]
```
